---
name: Project page caption spacing
description: Spacing and style for captions/text that appear directly below media elements on project pages
type: feedback
---

Use `margin-top: 0.75rem` for text/captions that sit directly below a media element (hero, video, image) on project pages. Keep text close to its media — don't use 2rem or larger gaps.

**Why:** User confirmed this spacing feels right after we tightened it from 2rem to 0.75rem.

**How to apply:** Whenever adding a new caption, one-liner, or label beneath a media block on a project page, default to `margin-top: 0.75rem`.
